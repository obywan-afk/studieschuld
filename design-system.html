Studieschuld · Design System v1

Bold Modern

A neo-grotesk, high-contrast system: true-black canvas, fully saturated color cards, black ink on color, heavy display type and tiny uppercase eyebrows. Built on the native system-ui font — no webfont, no tracking. This page is the single source of truth for the tokens below.
